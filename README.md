# 🚀 Featured Projects

## 🔥 DevGrill

**AI-powered mock interviews that feel like the real thing.**

DevGrill creates personalized technical and system design interviews using a candidate's resume and the job description they're targeting. Instead of asking generic questions, it focuses on identifying knowledge gaps and simulates realistic interview conversations.

### Features

- 🤖 Multi-agent interview system powered by LangGraph
- 📄 Resume + Job Description based personalization
- 💬 AI interviewer with contextual follow-up questions
- 🧠 System Design & Technical Interview modes
- 💻 Built-in coding environment
- 📊 Detailed feedback reports with strengths, weaknesses and actionable improvements
- 💾 Cross-device interview persistence
- 👤 Guest interviews without signup

### Tech Stack

- Next.js 15
- TypeScript
- Express.js
- PostgreSQL (Neon)
- LangGraph.js
- DeepSeek V4
- Claude
- Clerk Authentication
- Drizzle ORM
- Tailwind CSS
- Railway
- Vercel
